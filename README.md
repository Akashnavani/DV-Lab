## Contents

### PartB 1
Basic data visualization fundamentals and introductory visualization techniques.

### PartB 2
Solar efficiency temperature analysis with CSV data source (`solar_efficiency_temp.csv`).

### PartB 3
Student data analysis and visualization using the `student_data.csv` dataset.

### PartB 4
Sales data visualization and analysis using the `sales.csv` dataset.

### PartB 5
Advanced visualization techniques and data exploration methods.

### PartB 6
Income clustering analysis and machine learning visualization using the `income_clustering.csv` dataset.

## Requirements

- Python 3.x
- Jupyter Notebook
- pandas
- matplotlib
- seaborn
- scikit-learn
- numpy

## Installation

1. Clone the repository:
```bash
git clone https://github.com/Akashnavani/DV-Lab.git
cd DV-Lab
```

2. Install required packages:
```bash
pip install jupyter pandas matplotlib seaborn scikit-learn numpy
```

## Usage

1. Navigate to the project directory
2. Start Jupyter Notebook:
```bash
jupyter notebook
```

3. Open the desired notebook file (PartB1.ipynb through PartB6.ipynb) to view and run the code

## How to Run Individual Notebooks

Each notebook (PartB1.ipynb - PartB6.ipynb) can be run independently:
1. Open the notebook in Jupyter
2. Click "Kernel" → "Restart & Run All" to execute all cells
3. Or run cells individually using Shift+Enter

## Dataset Files

- `solar_efficiency_temp.csv` - Solar panel efficiency vs temperature data
- `student_data.csv` - Student performance and demographic data
- `sales.csv` - Sales transaction data
- `income_clustering.csv` - Income data for clustering analysis
